# Pluto

## Installation

**1.** Install gcc

-   Ubuntu:

        `sudo apt-get install gcc`

**2.** Install make

-   Ubuntu:
  
        `sudo apt-get install make`

**3.** Clone the github repository

    `git clone https://github.com/BenjaminBalaam/pluto.git`

**4.** Build the program with the makefile

    `make`

**5.** Run the program

-   Ubuntu:

        `./pluto example.ps`
