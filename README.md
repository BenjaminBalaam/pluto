# Pluto Script

## Installation

-   Install gcc
    Ubuntu:
    `sudo apt-get install gcc`

-   Install make
    Ubuntu:
    `sudo apt-get install make`

-   Clone the github repository
    `git clone https://github.com/BenjaminBalaam/pluto-script.git`

-   Build the program with the makefile
    `make`

-   Run the program
    Ubuntu:
    `./pluto-script example.ps`
